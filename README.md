# Project3
An analysis  of car sales from historical, car sales data to assist the new Head of Sales, to expand the dealership network, improve sales performance, and optimize car sales. 
# Purpose 
- Provide insights on car sales
- Determine the factors affecting car sales
- Give recommendations to to increase car sales
# Aim of the study
To perform an analysis of the SQL query with car sales data,to generate a detailed summary of buyers and sellers behaviour.
# Tools used
- Gantt chart on Microsoft Excel
- Data analysis planning on MIRO
- Snowflake to run SQL queries
- Microsaft excel to create Pivot tables and graphs
- Powerpoint presentation
- Github to load the final report
  # Data Sources
  - Bright car sales CVS file
  # Key findings
  -The automatic cars are most popular than manual cars. Since these cars are more easy to operate, it is suggested that latest car models incorporate latest AI (Artificial Intelligence)technology for easy operations.
  -The Sedans, SUV and Utility vans are most popular. Latest models could be more family and cormecial friendly.
  -The high performing cars are Ford, Chevrolet, Nissan which are durable and lasting than the latest cars.
  -A market research on preferances by car buyers would give a detailed report on the buyer's psychology when it comes to buying cars.
